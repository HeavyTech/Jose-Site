Projects

Machine Learning for Signal Quality Monitoring sponsored by AT&T

The purpose of this project was to develop a web service in conjunction with a mobile application to to handle push notifications regarding anomalies in signal quality for AT&T.

  • Accountable for creating a registration page with proper encryption for passwords.
  • Assisted in building the server side which allowed connections between the Android app and server.
  • Assisted in building queries that user will be able to use through the app.

Starship Fighter

Starship Fighter gameplay initializes the player at the bottom of the screen while enemy aircraft starts to follow you and fire at you. You are able to get hit 5 times before you starship fighter gets destroyed.This game was developed using Python along with the artwork of Kenney.Game includes extra guns, shields, bonus. Game was inspired by Galaga from the 80's.

Taxi Road Application.

Application will parse data from Open Street Map. It will calculate the fastest route to the given destination. In the example we had put two "Taxis " on the map and we were able to calculate which taxi had the fastest route.This application was done using Java, Eclipse, and BFS algorithm.
